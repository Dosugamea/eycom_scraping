# eycom_scraping
[破棄された案件] EycomをスクレイピングするGUIアプリ (兼 PySimpleGUIの練習)

## 実行方法
```
(前提ライブラリを入れた上で)
python main.py
```

## 使用ライブラリ
- requests
- lxml.html
- PySimpleGUI (tkinterのWrapper)
- PyInstaller (pyinstaller main.py --onefile --noconsole)
